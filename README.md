## 🚀 Features

### 🔐 Authentication
- Secure user login using **JWT authentication**
- Protected APIs (only logged-in users can access data)

### 😴 Sleep Tracking
- Add sleep & wake times
- Automatically calculate sleep duration
- View sleep trends on dashboard

### 🙂 Mood Tracking
- Log daily mood
- View mood distribution in charts
- Track emotional well-being over time

### 🏃 Exercise Tracking
- Add exercise type & duration
- View total and daily exercise activity

### 📊 Dashboard (Streamlit)
- One-page interactive dashboard
- Graphs & analytics
- Clean UI with sidebar navigation
